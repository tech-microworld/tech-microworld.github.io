# 快速上手

# 环境安装

  • 安装 etcd（必须）
  • 安装 OpenResty（必须）
  • 安装 test::nginx（开发需要，跑测试用例）
  • 安装 luarocks （必须，用来安装项目依赖）
  • 安装 luacheck（开发需要，代码风格检查）
    luarocks install luacheck
  • 安装 nodejs（开发控制面板需要）
  • 安装 yarn 或者 npm（开发控制面板需要）
    • npm（安装 node 自带）
    • yarn

环境变量

# 根据实际安装路径修改
export PATH="$PATH:/usr/local/Cellar/openresty/1.15.8.3_1/nginx/sbin"
export PATH="$PATH:/usr/local/Cellar/openresty/1.15.8.3_1/luajit/bin"

启动etcd & 修改配置文件 etcd 配置conf/app.json

etcd.http_host 为 etcd 服务访问地址，etcd.data_prefix 为全局 etcd key 前缀，如有必要可修改

{
    "appName": "ws-cloud-gateway",
    "env": "test",
    "etcd": {
        "http_host": "http://192.168.41.47:2379",
        "data_prefix": "/my/cloud/"
    }
}

# 启动服务

安装依赖

make deps

初始化数据（也可以跳过，在后台控制面板手动添加）

make init
