# rewrite plugin

url 重写

rewrite_plugin.lua

# 说明

rewrite 插件用于重写请求 url 再转发到上游服务

# 参数

参数名 类型 默认值 说明
rewrite_url_regex string nil 匹配正则
rewrite_replace string nil 替换字符串

通过 ngx.re.gsub(uri, rewrite_url_regex, rewrite_replace, "jo") 替换得到目标 url

# 示例

{
  "rewrite_url_regex": "^/innerapi/(.*)/",
  "rewrite_replace": "/"
}

当请求 /inner/demo1/info url 会重写为 /innerapi/info
