# discovery plugin

服务发现

discovery_plugin.lua

# 说明

服务发现需要配置路由和服务节点（程序启动自动注册或手动添加）

服务节点查询过程

+--------+     +--------+     +--------+      +--------+ 
| 请求URL |+--->| 服务名  |+--->| 服务节点 |+--->| 负载转发 |
+--------+     +--------+     +--------+      +--------+

# 参数

无

# 设置

  1. 添加【路由】 -> 【服务】映射

http://localhost:10000/dashboard/#/gateway/routes

路由协议支持 http 和 grpc

  1. 添加服务节点

大多数情况下，服务节点应该是在服务启动时自动注册，控制面板也提供了手动维护功能
